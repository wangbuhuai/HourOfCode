Introduction

The Hour of Code workshop is hosted by the Department of Workforce & Business Development of the St. Charles County (Missouri), and provided by the Department of Computer Science of the St. Charles Community College.  The workshop is free and open to the general public, and has two different sessions.

Introduction to Computer Programming Introduction to Object-Oriented Programming
This 90-minute session provides the attendees who do not have any programming background some hands on experience of developing computer programs with the Python programming language.  The attendees will also have an opportunity to understand some of the basic concepts, theories, techniques/approaches of computer programming, in order to let them better plan/prepare their future study. This 90-minute session provides the attendees who do not have any programming background some hands on experience of creating some objects and the logic/story amongst the objects using Alice 3.  The attendees will also have an opportunity to briefly understand the basic concepts, theories, and techniques of object-oriented programming, the only accepted programming style in the IT world today.

Knowledge/Skills Required for the Workshop

1)Attendees are expected to have basic keyboard and mouse skills.

2)Attendees are expected to have basic skills of using general-purpose software (e.g. website browser, document processor).

3)Attendees are expected to have some basic knowledge about computer files, and have basic skills of opening/saving files.

4)Attendees are not expected to have any background about computer programming.

Hardware/Software Required for the Workshop

1)Every attendee must have a personal computer, desktop or laptop.  Smartphones, tablets (e.g. iPad), or wearables (e.g. smartwatch) do not work.

2)Microsoft Windows, Linux, or Macintosh operating system should be installed in the computer.

3)The computer must have access to the Internet.

4)Webcam and microphone are highly recommended to be installed in your computer so that you can communicate with the lecturer during the workshop.

5)(Session of Introduction to Computer Programming only) Google Chrome (a website browser) must be installed on the computer.  Unfortunately, other browsers (e.g. Internet Explorer, Microsoft Edge, Mozilla Firefox, Safari, Opera) may not work.  The link below will direct you to download Google Chrome.

https://www.google.com/chrome

6)(Session of Introduction to Object-Oriented Programming only) Alice 3 should be installed in your computer.  You are highly recommended to have a computer with Microsoft Windows 10 installed on it, since the installation of Alice 3 on Windows 10 will be covered during the session.  If your operating system is not Windows 10, then you need to install Alice 3 by yourself prior to the session.  The link below will direct you to the download page of Alice 3.

https://www.alice.org/get-alice/alice-3
